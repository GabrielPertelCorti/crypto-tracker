# Crypto Tracker

Projeto simples de acompanhamento de criptomoedas desenvolvido com React.
A aplicação consome dados de uma API pública e exibe informações atualizadas sobre diferentes criptomoedas.

É possível visualizar dados como preço atual, valor de mercado e variação percentual, além de acessar uma página com detalhes de cada moeda.

## 📊 Funcionalidades
